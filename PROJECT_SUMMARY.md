# Hospitality Manager - Project Summary

## Overview
Hospitality Manager is a modern, frontend-only web application designed for hotels and hospitals to efficiently manage bookings, guests, and daily operations. Built with React and Vite, it features a clean, professional UI with a warm hospitality-focused color palette.

## Key Highlights

### Technology
- **Framework**: React 18.2 with Vite 5.0 for lightning-fast development
- **Styling**: Custom CSS design system with hospitality color palette
- **Data**: Google Sheets integration for cloud-based data storage
- **Icons**: Font Awesome for professional iconography

### Features
1. **Dashboard** - Real-time analytics showing bookings, guests, and revenue
2. **Booking System** - Comprehensive form for creating new reservations
3. **Guest Management** - Searchable, filterable directory of all guests
4. **Responsive Design** - Works perfectly on all devices
5. **Cloud Storage** - Data persists in Google Sheets via Apps Script

### Design Philosophy
- **Color Palette**: White, Beige, Soft Orange, Grey (no blue/purple)
- **Typography**: Playfair Display for elegance, Inter for readability
- **UI Elements**: Gradient accents, smooth animations, card-based layouts
- **Icons**: Font Awesome icons throughout (no emojis)

## Target Use Cases
- Hotel front desk operations
- Hospital patient management
- Guest house reservations
- Clinic appointment tracking

## Deployment
- Frontend-only application
- Can be hosted on any static hosting service (Netlify, Vercel, GitHub Pages)
- Google Sheets serves as the backend database
